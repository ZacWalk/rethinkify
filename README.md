Rethinkify
==========

Simple text editor for programmers and normal people. Windows only.

Few dependencies, simple Win32 GDI rendering, few features.

Multiple documents can be open at once within the current folder. Switching files does not prompt to save — modified documents stay in memory in the background with full undo history. Modified files are highlighted in red in the file list. You are only prompted to save when changing the root folder or exiting.
