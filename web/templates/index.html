My Projects

  • diffractor app

    Lightweight photo and video management for Windows. Optimised with Direct3D/HLSL and SIMD. I have been working on it for 20 years, and I am still not sure my photos are organised. It used to be called ImageWalker.

    github.com/diffractor

  • nnc

    For a while I was a compiler engineer — back when Microsoft paid me to write the Windows-on-ARM backend for GCC. That was a lot of fun, and I always had this idea that if you could just compile an LLM for CPU, then Jensen Huang would be out of business. So this is my attempt. It can compile and run Gemma 3 at about 200 ms per token on my PC with AVX2. Jensen Huang is safe for the moment, at least until the CPU vendors get their act together and add a SIMD instruction set for FP8.

    github.com/ZacWalk/nnc

  • spike-db

    A time-series database designed for financial market data. This is used in my trading experiments and Alpha Galactica. The unique feature is very fast in-order iteration of market data. Useful if you need to do that sort of thing.

    github.com/ZacWalk/spike-db

  • dos4z

    A while back, with the help of Scott Hanselman, MS-DOS 4.0 was made open source. I had the idea of adding large hard drive support so that my IBM 5150 could use bigger disks. Building it is tricky because it relies on some bundled DOS-only compilers and assemblers, so I wrote my own DOS shim that lets those DOS programs run on a modern Windows machine. You can now build this version of DOS on a modern machine, and my IBM 5150 will boot from the result. I still have not figured out how to add the large disk support though.

    github.com/ZacWalk/dos4z

  • electron-bench

    Measuring messaging cost across Electron communication paths. I originally wrote this when I worked on Microsoft Teams to improve performance. Teams does not use Electron any more, but if you do, this might be helpful.

    github.com/ZacWalk/electron-bench

  • rust-http-sys

    A few years ago I started working with Rust at Microsoft, with the goal of improving performance for some important services that make Office work. Those services are used by hundreds of millions of people, so making them more efficient adds up to a lot of money saved. One of my initial tasks was to prototype this wrapper for http.sys, the Windows kernel-mode web server. I was quite proud of it, but one of the Rust experts told me it was written like a C++ developer trying to write Rust — and he rewrote it all. Not surprising really, because I am a C++ guy trying to write Rust.

    github.com/ZacWalk/rust-http-sys

  • stuntcarracer

    I used to love Stunt Car Racer when I was a kid on my Commodore 64. I noticed that the code from the Amiga version was floating around, so I got a copy and rewrote it to use floating-point math instead of fixed-point. The fact that the 3D graphics were fully implemented in fixed-point math shows how amazing those old-school programmers were. On modern hardware, floating point has no real performance penalty any more, but unfortunately I introduced a bug where you occasionally fall through the track. I am stumped by it — even Claude Opus 4.7 could not diagnose the root cause. Maybe I will change it all back to fixed point if I get the time.

    github.com/ZacWalk/stuntcarracer

  • low-ban

    Around 2015 I made low-ban as a hackathon project at Microsoft. At the time I was working on Skype. The basic concept was a video feed that could work on a 2G network, around 10 kbps. At that speed you can only send a blurry mess, so I used facial landmarks to transmit coordinates representing facial expressions. That way you could at least see whether the person was smiling. There was some interest, and I even presented it to the head of design for Skype — he wore black and really looked the part. In the end nothing came of it, and now we all have enough bandwidth for high-resolution video all the time.

    github.com/ZacWalk/low-ban

  • system-stuff

    Way back around 2000, the company I worked for still supported Windows 95 and 98 for the desktop application we sold. I usually volunteered to test on those platforms, and I had written my own task-manager-like program. The Windows 95 task manager is very basic, and I thought I could do better. Every five years or so I get it building again and make a few updates. It was originally called sys-mate, and is now called more generically system-stuff.

    github.com/ZacWalk/system-stuff

  • noterad

    github.com/ZacWalk/noterad

  • ergo-active

    Back in the late 90s my neighbour in London, a Swedish furniture designer, came up with the idea for Ergo Active over beers. The premise was simple: remind you to take a break when you have been using the computer for too long. We had so much fun making the app, and filled it with information starting with “experts recommend…” Once it was finished we were not sure what to do next — there was no app store in those days. Luckily it is preserved here. If you forget to take breaks, this is the app for you.

    github.com/ZacWalk/ergo-active

  • nn

    Fashion-MNIST is a dataset of pictures of clothes. It is used for a machine learning challenge to train a model to recognize the different items. As a learning exercise, I wrote my own multi-layer perceptron to understand how deep learning is implemented. Anyway, my implementation does as well as any other multi-layer perceptron, and I was able to optimize it for modern SIMD hardware, so it is fast, for a CPU. If you need to tell the difference between a sock and a shirt, this is the program!

    github.com/ZacWalk/nn

  • potato

    Back in the 2000s I needed to move from C++ to work on a web project for a while. I wanted to learn CSS better, so I decided the best thing to do was write my own web browser in C++. Here is what I came up with — probably the worst web browser in the world. I try to keep it working well enough to display Wikipedia. It is not working right now because Wikipedia keeps adopting more modern CSS, but it works fine on Wikipedia from 2015 — so super useful if you ever go back in time.

    github.com/ZacWalk/potato
